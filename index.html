Welcome to IDRIX fork of wxFormBuilder

At IDRIX, we build and sign our own wxFormBuilder Windows installer since the official installers are not signed and thus can not be trusted. Moreover, the current version of the official sources contains an issue when generating C++ code of wxTextCtrl control which made us implement a fix to make it usable on our use cases.

You can download the latest Windows installer (version 3.5.2-RC2) from the link above.
